   <tt class="descname">exec_cmd</tt><big>(</big><em>cmd</em>, <em>**kwargs</em><big>)</big><a class="reference internal" href="../_modules/JumpScale/lib/cisco_ios/Router.html#Router.exec_cmd"><span class="viewcode-link">[source]</span></a><a class="headerlink" href="#JumpScale.lib.cisco_ios.Router.Router.exec_cmd" title="Permalink to this definition">¶</a></dt>
   <dd><p>Execute Command</p>
   <p>Arguments:
   cmd                  &#8211; String. Command to be executed</p>
   <blockquote>
   <div>(Mandatory)</div></blockquote>
   <dl class="docutils">
   <dt>expects              &#8211; List. List of possible matches</dt>
   <dd>(Default to [])</dd>
   <dt>return_output        &#8211; Boolean. If True, returns Output</dt>
   <dd>(Default to True)</dd>
   <dt>return_matched        &#8211; Boolean. If True, returns matched expect</dt>
   <dd>(Default to False)</dd>
   <dt>prompt               &#8211; String. Expected Prompt after</dt>
   <dd>command execution (Default to None)</dd>
   <dt>auto_complete        &#8211; Boolean. If True, hits spaces</dt>
   <dd>till prompt is seen (in case of huge output)
   (Default to True)</dd>
   </dl>
   <p>Example:
   login(&#8220;Telnet R1&#8221;)                       # Go to User Exec Mode
   exec_cmd(&#8220;show clock&#8221;)                   # Returns Output
   exec_cmd(&#8220;enable&#8221;,</p>
   <blockquote>
   <div>expects=[&#8220;Password:&#8221;, &#8220;R1#&#8221;],
   prompt=&#8221;R1#&#8221;)                       # Go to Enable Mode</div></blockquote>
   <p>exec_cmd(&#8220;Conf t&#8221;, prompt=&#8221;R1(Config)#&#8221;)   # Go to Config Mode</p>
   </dd></dl>
